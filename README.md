# Contact App

Welcome to the **Contact App** repository! This app is designed to be the ultimate contact management solution, offering users an intuitive and feature-rich experience.

## Features
- **Modern UI/UX**
- **Advanced Search**
- **Customizable Contact Fields**
- **Backup & Restore**
- **Privacy-Focused**

## Tech Stack
- **Kotlin**
- **Jetpack Compose**
- **Room Database**
- **Coroutines & Flows**
- **Material Design**
